Here’s what you can ask about. I have access to the following datasets:

*payments*
This table contains information about the payments made by users for their subscriptions.
You can ask about: payment_id, subscription_id, payment_date, amount_usd, method, total_revenue_usd, total_payments

*sessions*
This is a table that contains information about the type of activity and duration of the sessions for users.
You can ask about: session_id, user_id, session_date, duration_minutes, activity_type

*subscriptions*
This table contains information about the subscriptions of users to different plans.
You can ask about: subscription_id, user_id, plan, start_date, end_date, status, subscription_length_days

*users*
This table contains information about the users of the app, the device they signed up on, and the country of signup.
You can ask about: user_id, signup_date, country, device_type
